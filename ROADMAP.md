# Roadmap

## v0.1

- Deterministic local lifecycle.
- Failure-path demos for cash failure, restricted buyer, and coupon resume.
- Hash-chained audit verification.
- Forkable signer, cash leg, and registry interfaces.

## v0.2

- Foundry fuzz tests for DvP state transitions.
- Secondary-transfer scenario with lockup expiry.
- Optional Postgres compose profile.

## v0.3

- Net settlement design prototype.
- ISO 20022-shaped event export from settlement service.

## v0.4

- Second local chain profile.
- Pluggable console views for additional settlement models.

Maintained as a portfolio reference. Issues and pull requests are reviewed when time allows.
